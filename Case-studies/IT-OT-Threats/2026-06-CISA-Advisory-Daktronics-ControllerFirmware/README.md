# Case Study: Daktronics Controller Firmware Multi-CVE Root-Access Chain (ICSA-26-176-04)

**Category:** IT-OT-Threats
**Source:** CISA ICS Advisory
**Date Published:** Week of 2026-06-22
**Risk Rating:** High
**Bundle Date:** 2026-06-29

## Summary
CISA published ICS Advisory ICSA-26-176-04 describing a chain of vulnerabilities (CVSS up to 9.3) in Daktronics display controller firmware that together allow root-level device access. No confirmed active exploitation at this time, but the severity and multi-sector deployment footprint — including transportation-adjacent and public-safety signage — warrant full assessment, consistent with program precedent (RR-020, Rockwell FactoryTalk Historian).
